# Lessons Learned

**Every agent MUST read this before build, deploy, or audit sessions.**
**Any agent CAN add a lesson during their session.**

---

## Getting Started
- **Read dispatch files BEFORE doing anything.** If you skip dispatch, you risk contradicting what another agent already decided. This is the #1 operational mistake.
- **Save findings to memory/ immediately.** Sessions are ephemeral — if it's not in a file, it didn't happen.
- **Health checks verify infrastructure, not user experience.** Tables existing and env vars being set does NOT mean users can complete the core flow. Test the actual user journey.
