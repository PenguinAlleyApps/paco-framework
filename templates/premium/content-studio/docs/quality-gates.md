# Quality Gates

## Before Delivering Any Output
- [ ] Meets the brief / requirements
- [ ] No false claims or unverified data
- [ ] Proofread and professional quality
- [ ] Dispatch updated with activity and handoffs

## Before Deploy (if applicable)
- [ ] Health check passes
- [ ] Core user flow tested end-to-end
- [ ] No credentials in committed code
- [ ] Security headers present
